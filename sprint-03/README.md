Решения задач из третьего третьего спринта курса «Алгоритмы и структуры данных». В этом модуле фокус сделан на изучении рекурсивных подходов, алгоритмов перебора, различных методов сортировки (от простых до $O(N \log N)$) и модификаций бинарного поиска.

Условия задач включены в начало каждого `.py` файла в виде docstring. Код написан в соответствии со стандартом PEP8.

### 📘 Основные задачи

| Файл | Описание |
| :--- | :--- |
| `a_bracket_generator.py` | Генератор правильных скобочных последовательностей через рекурсию. |
| `b_combinations.py` | Генерация всех возможных комбинаций букв, соответствующих цифрам на кнопках телефона. |
| `с_subgen.py` | Эффективная проверка одной строки на вхождение в другую в качестве подпоследовательности. |
| `g_wardrobe.py` | Сортировка предметов трех цветов (0, 1, 2) за один проход методом подсчета. |
| `h_big_number.py` | Формирование максимально возможного числа из набора строк с использованием кастомной логики сравнения. |
| `j_bubble.py` | Реализация сортировки пузырьком с оптимизацией (досрочный выход) и логированием этапов. |
| `k_merge_sort.py` | Классическая сортировка слиянием (merge sort) с разделением массива на подзадачи. |
| `l_two_bicycles.py` | Поиск минимальных дней для покупки велосипедов с помощью итеративного бинарного поиска. |
| `l_two_bicycles_recursion.py` | Рекурсивная версия бинарного поиска для нахождения индексов в отсортированном массиве. |
| `n_flower_beds.py` | Алгоритм объединения перекрывающихся временных или пространственных интервалов. |
