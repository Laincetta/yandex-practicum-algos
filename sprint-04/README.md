# 🏁 Спринт 4 — Хеширование и структуры данных

Решения задач из четвертого спринта курса «Алгоритмы и структуры данных».

## 📘 Основные задачи

| Файл | Описание |
| :--- | :--- |
| `a_groups.py` | Фильтрация уникальных названий групп через хеширование |
| `a_monitoring.py` | Транспонирование матрицы $m \times n$ |
| `b_competition.py` | Поиск самого длинного отрезка с равным количеством 0 и 1 |
| `b_todo_list.py` | Вывод элементов связного списка |
| `c_least_favorite_thing_todo.py` | Удаление узла из односвязного списка по индексу |
| `c_strange_comparison.py` | Проверка изоморфизма двух строк по паттерну |
| `d_caring_mom.py` | Поиск индекса значения в связном списке |
| `d_polynom_hash.py` | Реализация полиномиального хеша для строк |
| `e_break_me.py` | Скрипт для генерации коллизий в хеш-функции |
| `e_opposite.py` | Разворот двусвязного списка |
| `f_stack_max.py` | Стек с поддержкой максимума |
| `f_stack_max_effective.py` | Оптимизированный стек с получением максимума за $O(1)$ |
