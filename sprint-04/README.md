# 🏁 Спринт 4 — Хеш-функции

Решения задач из четвёртого спринта курса «Алгоритмы и структуры данных». Основные темы: устройство и применение хеш-таблиц, методы разрешения коллизий (метод цепочек), разработка поисковых индексов и работа со строковыми хешами.

Код соответствует стандарту PEP8. Условия задач и подробный анализ временной/пространственной сложности приведены в комментариях к каждому файлу.

### 📘 Основные задачи

| Файл | Описание |
| :--- | :--- |
| `a_groups.py` | Фильтрация уникальных названий групп с использованием хеш-таблицы (аналог set). |
| `b_competition.py` | Поиск максимального отрезка с равным количеством результатов «0» и «1» через префиксные суммы и хеширование. |
| `c_strange_comparison.py` | Проверка изоморфизма двух строк: определение взаимно однозначного соответствия символов. |
| `d_polynom_hash.py` | Реализация алгоритма полиномиального хеширования строки по схеме Горнера. |
| `e_break_me.py` | Скрипт для генерации коллизий: поиск разных строк с одинаковым значением хеш-функции. |
