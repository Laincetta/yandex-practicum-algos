# 🏁 Спринт 6 — Графы

Решения задач из шестого спринта, посвященного структурам данных на графах. В фокусе: способы представления графов (матрицы и списки смежности), алгоритмы обхода (DFS, BFS), поиск компонентов связности, топологическая сортировка и алгоритмы построения остовных деревьев.

Проекты охватывают как классические задачи на связность, так и оптимизационные задачи на взвешенных графах.

### 📘 Основные задачи

| Файл | Описание |
| :--- | :--- |
| `A_edge_list_to_adj_list.py` | Построение списка смежности из списка ребер ориентированного графа. Подсчитывает исходящую степень каждой вершины. |
| `B_edge_list_to_adj_matrix.py` | Генерация матрицы смежности для ориентированного графа, где `1` на пересечении означает наличие ребра. |
| `C_DFS.py` | Реализация классического обхода в глубину (DFS) с использованием трех цветов и сортировкой соседей для стабильного порядка вывода. |
| `E_connected_comps.py` | Поиск всех компонентов связности в неориентированном графе. Каждая вершина помечается ID своей компоненты. |
| `H_time_to_leave.py` | Модифицированный DFS для вычисления временных меток входа (`entry`) и выхода (`leave`) каждой вершины в процессе обхода. |
| `J_topological_sort.py` | Топологическая сортировка ориентированного ациклического графа (DAG) через инвертированный порядок времени выхода из DFS. |

### 🏆 Финальные задачи

| Файл | Описание |
| :--- | :--- |
| `final_a_water_world.py` | **Мир воды.** Подсчет количества островов и поиск площади самого большого из них. Используется итеративный BFS для обхода сетки, что экономит память и предотвращает переполнение стека. |
| `final_b_expensive_network.py` | **Дорогая сеть.** Реализация алгоритма Прима для построения максимального остовного дерева (MST). Используется приоритетная очередь (`heapq`) для эффективного выбора самого тяжелого ребра. |
