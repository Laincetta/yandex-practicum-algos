# 🏁 Спринт 5 — Деревья

Решения задач из пятого спринта, посвященного иерархическим структурам данных. В фокусе: бинарные деревья поиска (BST), сбалансированные деревья, приоритетные очереди и реализация пирамидальной сортировки (Heap Sort).

Проекты включают как базовые операции (вставка, удаление, обход), так и более сложные алгоритмы манипуляции структурами.

### 📘 Основные задачи

| Файл | Описание |
| :--- | :--- |
| `e_search_tree.py` | Проверка дерева на соответствие свойствам бинарного дерева поиска (валидатор BST). |
| `i_different_search_trees.py` | Вычисление количества различных уникальных структур BST, которые можно построить из N узлов (Числа Каталана). |
| `j_add_node.py` | Реализация алгоритма вставки нового узла в бинарное дерево поиска с сохранением его свойств. |
| `k_print_range.py` | Эффективный вывод всех ключей дерева, попадающих в заданный диапазон `[L, R]`. |
| `l_sift_down.py` | Реализация ключевой операции кучи — просеивания элемента вниз для поддержания свойств max-heap. |

---

### 🏆 Финальные задачи

| Файл | Описание |
| :--- | :--- |
| `final_a_heap_sort.py` | Пирамидальная сортировка (Heap Sort). Реализована через построение кучи (heapify) и последовательное извлечение максимума. Используется для ранжирования участников соревнований. |
| `final_b_delete_node.py` | Алгоритм удаления узла из BST. Обрабатывает три случая: узел-лист, узел с одним потомком и узел с двумя потомками (поиск преемника/successor). |
