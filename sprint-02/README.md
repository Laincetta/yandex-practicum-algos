# 🏁 Спринт 2 — Основные структуры данных

Решения задач из второго спринта курса «Алгоритмы и структуры данных». В этом модуле фокус сделан на реализации стеков, очередей, связных списков и деков.

Условия задач включены в начало каждого `.py` файла в виде docstring. Код — в стиле PEP8.

---

## 📘 Основные задачи

| Файл | Описание |
| :--- | :--- |
| `a_monitoring.py` | Транспонирование матрицы |
| `b_todo_list.py` | Вывод элементов односвязного списка |
| `c_least_favorite_thing_todo.py` | Удаление узла из односвязного списка |
| `d_caring_mom.py` | Поиск элемента в односвязном списке |
| `e_opposite.py` | Реверс двусвязного списка |
| `f_stack_max.py` | Стек с поддержкой операции получения максимума |
| `f_stack_max_effective.py` | Оптимизированный стек (get_max за O(1)) |
