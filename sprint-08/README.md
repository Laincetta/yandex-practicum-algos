# 🏁 Спринт 8 — Алгоритмы на строках

Восьмой спринт посвящен методам обработки текстовой информации. Рассматриваются как базовые операции (разворот, вставка, сравнение), так и продвинутые структуры данных, такие как **префикс-функция** и **бор (trie)**, позволяющие эффективно искать подстроки и проверять слова по словарю.

---

### 📘 Основные задачи

| Файл | Описание |
| :--- | :--- |
| `a_line_reversal.py` | **Разворот строки.** Простая задача на разбиение строки по пробелам и вывод слов в обратном порядке. |
| `b_border_control.py` | **Пограничный контроль.** Проверка, можно ли получить одну строку из другой ровно одной операцией: вставкой, удалением или заменой символа. |
| `e_lines_inserting.py` | **Вставка строк.** Эффективная сборка новой строки путем вставки заданных подстрок в определенные позиции исходного текста с использованием словаря вставок. |
| `g_shift_search.py` | **Поиск со сдвигом.** Поиск шаблона в массиве чисел, где совпадением считается сохранение постоянной разности (сдвига) между соответствующими элементами. |
| `k_compare_strings.py` | **Сравнение строк.** Нестандартное сравнение двух строк, учитывающее только те буквы, чья позиция в алфавите четная. |
| `l_prefix_func.py` | **Префикс-функция (наивная).** Реализация вычисления длин наибольших собственных префиксов через вложенные циклы и срезы. |
| `l_prefix_func_optimal.py` | **Оптимизированная префикс-функция.** Реализация за $O(n)$ с использованием динамического программирования для быстрого пересчета значений без лишних сравнений подстрок. |

---

### 🏆 Финальные задачи

| Файл | Описание |
| :--- | :--- |
| `final_a_packed_prefix.py` | **Распаковка строк и LCP.** Алгоритм распаковки строк вида `n[A]` с использованием стека для обработки вложенности и последующим поиском наибольшего общего префикса (LCP). |
| `final_b_cheat_sheet.py` | **Шпаргалка.** Проверка возможности разбиения текста на слова из словаря с помощью **бора (trie)** и **динамического программирования**. |
