# 🏁 Спринт 7 — Жадные алгоритмы и динамическое программирование

Этот спринт посвящен методам оптимизации. В первой части рассматриваются жадные стратегии, где локально оптимальный выбор ведет к глобальному максимуму. Во второй — динамическое программирование: решение сложных задач через разбиение на перекрывающиеся подзадачи.

---

### 📘 Основные задачи

| Файл | Описание |
| :--- | :--- |
| `A_exchange.py` | **Биржа.** Жадный алгоритм для максимизации прибыли от покупки и продажи акций. Суммируем все положительные разницы между соседними днями. |
| `B_timetable.py` | **Расписание.** Классическая задача на жадность: выбор максимального количества непересекающихся событий путем сортировки по времени окончания. |
| `C_gold_rush.py` | **Золотая лихорадка.** Жадный выбор наиболее ценных грузов (цена за кг) для заполнения рюкзака ограниченной емкости. |
| `F_stairs_jumps.py` | **Прыжки по лестнице.** DP-задача на поиск количества способов добраться до ступеньки $N$, если можно прыгать на расстояние от $1$ до $k$. |
| `H_flower_field.py` | **Цветочное поле.** Поиск маршрута с максимальным сбором ресурсов на сетке (движение только вправо и вверх). Базовое 2D DP. |
| `I_complex_field_with_flowers.py` | **Сложное поле.** Расширенная версия задачи H, включающая восстановление самого пути (строка команд 'U' и 'R'). |
| `K_horoscopes.py` | **Гороскопы.** Нахождение самой длинной общей подпоследовательности (LCS) для двух массивов с восстановлением индексов. |
| `L_leprechaun_gold.py` | **Золото лепреконов.** Вариация задачи о рюкзаке (Knapsack): поиск максимального веса, который можно набрать из золотых слитков. |

---

### 🏆 Финальные задачи

| Файл | Описание |
| :--- | :--- |
| `final_a_levenshtein_dist.py` | **Расстояние Левенштейна.** Оптимизированный алгоритм поиска редакционного расстояния между строками. Использует всего две строки массива вместо полной матрицы, снижая память до $O(m)$. |
| `final_b_equal_sums.py` | **Равные суммы.** Решение задачи о разбиении множества (Partition Problem). Определяет, можно ли разделить числа на две группы с равной суммой. Реализовано через одномерное DP. |
